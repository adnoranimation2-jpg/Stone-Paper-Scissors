# Stone-Paper-Scissors (C++)

A robust, console-based implementation of the classic Stone-Paper-Scissors game. This project demonstrates basic C++ concepts like enums, loops, and conditional logic with a focus on user experience.

## 🚀 Features
* **Multi-round Gameplay:** Choose how many rounds you want to play in one session.
* **Color-Coded Feedback:** The terminal changes color based on the result:
    * 🟢 **Green:** You Win!
    * 🔴 **Red:** You Lose.
    * 🟡 **Yellow:** It's a Draw.
* **Input Validation:** Handles both uppercase and lowercase inputs (e.g., "stone" or "Stone").
* **Replayability:** Option to restart the game immediately after finishing rounds.

## 🛠️ How to Run
1. Ensure you have a C++ compiler installed (like MinGW or GCC).
2. Download the `Stone-Paper-Scissors.cpp` file.
3. Compile using the terminal:
   ```bash
   g++ Stone-Paper-Scissors.cpp -o Stone-Paper-Scissors
4. Run the executable:
``` ./Stone-Paper-Scissors ```

## Requirements
OS: Windows (uses windows.h for console colors).

Language: C++
